jko will execute someCommand with arg1 ... argN as arguments, where arg1 is the first argument and argN is the last argument.

info

jko will create an environment variable for each argument passed, named jko_arg_#, where # starts at 0.
For example, jko_arg_0 will contain the value of the first argument, and jko_arg_N_1 will contain the value of the last argument.
arg1 saved in jko_arg_0
arg2 saved in jko_arg_1
arg3 saved in jko_arg_2
...and so on.

tip
  • Always verify the existence and validity of imported files (Error 10).
